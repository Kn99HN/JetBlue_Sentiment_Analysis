jetBlue Sentiment Analysis Project

This project is Khanh and Aron's attempt to understand the customers' reviews about jetBlue Airways. We wanted to see what the general sentiment around customers who do review online about jetBlue. We found that they tend to fall on either extreme(super negative or positive), which is expected given the polarization trend on the internet. We use Python and its various libraries to parse and process the data. We use Plotly libraries to create interactive plots to see the trends of the customers' feedback, and we created this webpage to view these plots.
